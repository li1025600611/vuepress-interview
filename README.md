# 速学 webpack5

根据个人学习经验编写，快速拿捏 webpack5

在线阅读地址: [https://rhyszhao.github.io/webpack-learn/](https://rhyszhao.github.io/webpack-learn/)

持续更新中...

## 克隆到本地阅读

### 克隆

```shell
git clone https://github.com/RhysZhao/webpack-learn.git
```

### 安装

```shell
npm install
```

## 启动

```shell
npm start
```
